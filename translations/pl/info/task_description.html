Aby zidentyfikować cztery punkty, które tworzą kwadrat, proszę zauważyć, że każdy kwadrat ma lewy dolny punkt narożny x, y i wektor kierunku dx, dy w kierunku jego lewego górnego punktu narożnego, który spełnia dx>= 0 & dy> 0, więc punkty odpowiednio lewego górnego, prawego dolnego i prawego górnego rogu tego kwadratu są również zawarte w punktach {% if is_js %}[x+dx,y+dy], [x+dy,y-dx], [x+dx+dy,y-dx+dy]{% else %}(x+dx,y+dy), (x+dy,y-dx), (x+dx+dy,y-dx+dy){% endif %} dla odpowiednio lewego górnego, prawego dolnego i prawego górnego rogu tego kwadratu są również zawarte w punktach. Można zatem (efektywnie) iterować przez wszystkie możliwości dla lewego dolnego punktu i wektora kierunku i mieć gwarancję znalezienia wszystkich kwadratów w siatce.
