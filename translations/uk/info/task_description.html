За заданою послідовністю точок (координати x, y як невід'ємні цілі числа) ваша функція повинна підрахувати, скільки існує таких квадратів, у яких усі чотири кути належать точкам. Зверніть увагу, що ці квадрати не обов'язково повинні бути вирівняні по осям, тобто їх сторони повинні бути або горизонтальними, або вертикальними. Наприклад, точки {% if is_js %}[0, 3], [3, 0], [6, 3], [3, 6]{% else %}(0, 3), (3, 0), (6, 3), (3, 6){% endif %} визначають квадрат, навіть якщо з нашої вирівняної по осях точки спостереження він виглядає як ромб.

До речі, якщо ви хочете спробувати інший підхід і порахувати квадрати, маючи лінії між точками, The Square Chest чекає на вас!
